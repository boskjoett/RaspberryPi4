# RaspberryPi4
This repo contains Raspberry Pi 4 projects written in C# using .NET Core.

| Folder | Content |
|--------|---------|
| led-flasher | Simple program that flashes an LED connected between GND (pin 9) and GPIO 17 (pin 11) |
